# Património Familiar (PWA)

App web offline-first para registo de património, distribuição, rendimento passivo e movimentos.

## Importar movimentos do banco (Excel)
Na tab **€**, usa **Importar movimentos (Excel do banco)** e carrega o ficheiro mensal (.xls/.xlsx).
A app tenta reconhecer as colunas e ignora duplicados automaticamente.
